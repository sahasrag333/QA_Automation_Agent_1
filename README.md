# ai-qa-agent

Project scaffold for an AI-powered QA automation agent.

Structure:
- `recorder/` - UI or session recording helpers
- `agents/` - agents that perform tasks (BDD, code-gen, framework manager)
- `framework/` - test framework integrations
- `recordings/` - saved recording artifacts
- `generated/` - generated test code and artifacts
- `reports/` - test reports

Run:

Use Python 3.8+ and install dependencies from `requirements.txt`.

python main.py
